# road-anomaly-detection

Overview

This project was developed as a university final project for detecting and analyzing road anomalies using sensor, GPS, and image data.

Project Contents

- Python scripts for data processing and analysis
- Road images dataset
- GPS and sensor data files
- Excel files containing processed results
- Final project report (PDF)

Technologies Used

- Python
- Excel
- GPS Data Processing
- Image Analysis

Files

- "1-1.py" to "1-4.py" : Main analysis scripts
- "road_images/" : Road image dataset
- "sensor_data*.xlsx" : Sensor datasets
- "gps_gpgga_data5.xlsx" : GPS data
- "final_report.pdf" : Final project report
